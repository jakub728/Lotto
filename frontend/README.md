# LOTTO App

## 📝 About

If you want to check the latest EuroJackpot results and generate your own numbers using an advanced number generator, feel free to try your luck! 😊


## ✨ Technologies

- Built with **React (JavaScript)**
- Divided into **front-end** and **back-end** components
- Back-end developed using **Express**, along with **fs** and **path** modules
- Lotto results fetched from **LOTTO OpenAPI** on the server side
- Advanced number generation powered by the **Chance** library
- Saving mode using **Local Storage** (for now 🚀), with unique IDs generated via **uuid**
- User-friendly alerts styled with **Toastify**
- Fully responsive — works on both **desktop** and **mobile**
- Android version currently in development 📱
